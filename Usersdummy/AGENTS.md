## Java Method Comment Rule

모든 Java 메서드를 생성하거나 수정할 때는 반드시 메서드 바로 위에 JavaDoc 형식의 주석을 작성한다.

주석 형식은 반드시 아래 규칙을 따른다.

```java
/**
 * @title 메서드의 목적을 한 문장으로 설명
 * @param 파라미터명 파라미터가 어디에 사용되는 값인지 설명
 * @return 반환값이 무엇인지 설명
 */
```

- `@title`은 모든 메서드 JavaDoc에 반드시 작성한다.
- 파라미터가 있는 메서드는 각 파라미터마다 `@param`을 작성한다.
- 반환값이 있는 메서드는 `@return`을 작성한다.
- `void` 메서드는 `@return`을 작성하지 않는다.
- 생성자는 메서드와 동일하게 바로 위에 JavaDoc을 작성하되, `@return`은 작성하지 않는다.
- 클래스 설명 JavaDoc은 필요할 때 작성할 수 있지만, 메서드 JavaDoc 규칙을 대체하지 않는다.
